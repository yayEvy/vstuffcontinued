# VStuff!

## Features as of v0.3.0

- Ropes!
- Rope Cutter, for cutting ropes
- A menu to select different styles of ropes (key can be changed)
- Rope coloring, by right-clicking a wool or plain rope with dye
- Phys Pulley, can be used to control rope lengths! (Reworked in 0.3.0)
- Mechanical Thruster (Will have some improvements in 0.3.1!)
- Phys Grabber! (Will also have improvements in 0.3.1, such as distance scrolling and better grabbing in general!)
- Pixel consistent rope textures
- Reaction Wheel (Originally started in Create: Propulsion)
- Levituff
- Rope Arrow
- Data driven ropes and rope styles (datapacks!)
- Experimental rope knot rendering (Is off by default, can be enabled in the config)
