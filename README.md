# VStuff!

## Features as of v0.3.0

- Ropes!
- Rope Cutter, for cutting ropes
- A menu to select different styles of ropes (key can be changed)
- Rope coloring, by right-clicking a wool or plain rope with dye
- Phys Pulley, can be used to control rope lengths! (Reworked in 0.3.0)
- Mechanical Thruster (Will have some improvements in 4.0!)
- Phys Grabber! (Has improvemens now, such as distance scrolling and better grabbing in general!)
- Pixel consistent rope textures
- Reaction Wheel (Originally started in Create: Propulsion, and also not fully implemented yet)
- Levituff (Slightly bugged in 3.0.1)
- Rope Arrow
- Data driven ropes and rope styles (datapacks!)
- Experimental rope knot rendering (Is off by default, can be enabled in the config, Will also be in 4.0 but still disabled, though possiblity improved. Will look pretty cool for phys ropes)
  
3.1.0 will most likely not happen as we have planned to just put that stuff into 4.0.0.
